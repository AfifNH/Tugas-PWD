3. HTML Tag merupakan tag untuk membuat sebuah dokumen html baru.

4. Head merupakan tag untuk membuat informasi tentang dokumen.

5. Title merupakan tag untuk membuat judul dari sebuah halaman.

6. Tag br merupakan tag untuk membuat baris kosong atau spasi dari sebuah halaman.

7. Tag center merupakan tag untuk meletakkan posisi baris menjadi di tengah-tengah halaman.

8. Tag p merupakan tag untuk membuat paragraf dari sebuah halaman.

9. Tag b merupakan tag untuk mempertebal huruf.

10. Tag i merupakan tag untuk memiringkan huruf.

11. Tag u merupakan tag memberi garis bawah pada huruf.

12. Tag q merupakan tag untuk memberikan sebuah kutipan pendek.

13. Tag hr merupakan tag untuk memisahkan konten atau baris dengan sebuah garis.

14. Body merupakan tempat dibuatnya semua konten website menggunakan HTML

15. Tag big merupakan tag untuk membuat text berhuruf besar

16 Tag small merupakan untuk membuat text berhuruf kecil

17. Tag sub merupakan tag untuk membuat tata penulisan huruf yang ditempatkan di bawah ketinggian huruf biasa, dan umumnya dalam ukuran lebih kecil dari huruf biasa. Subscript

18. Tag sup merupakan tag untuk membuat tata penulisan huruf yang terletak di sebelah atas huruf.Superscript

19. Tag s merupakan tag untuk mencoret text atau huruf yang tidak benar

20. Tag a href merupakan tag untuk membuat sebuah hyperlink yang mengarah ke website orang lain maupun file html yang lain
